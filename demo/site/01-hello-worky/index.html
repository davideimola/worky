Hello Worky

✅ Chapter 00 complete! Now write your first greeting.

What you'll do

Create a file called hello.txt containing the exact text Hello, Worky!. This simulates the kind of content-validation check a real workshop might do — for example, verifying that a config file has been correctly edited.

Checks

  • 📄
    hello.txt exists
    The file must exist in the current directory.
  • 🔍
    hello.txt contains "Hello, Worky!"
    The exact string must appear somewhere in the file.

Steps

# Create the greeting file
echo "Hello, Worky!" > hello.txt

# Run the checks
go run . check

Once the checks pass, chapter 02 — Finishing Up will unlock.

💡 Hint — exact string matters ▶

The check uses strings.Contains, so the greeting just needs to appear anywhere in the file. You could add more text after it and the check will still pass.
